#1 Mail Men

Mail Men is a Python/Django web application designed for efficient email management. Using Django Rest Framework and SQLite, it supports secure user authentication, a versatile interface for email handling, and advanced search functionalities. The app also features profile picture uploads and detailed email tracking. Deployed on AWS EC2 .

#2 Quick Chat

Quick Chat is a Python/Django web app that provides a streamlined tweet management system. Using Django Rest Framework and SQLite, it offers secure user authentication, a user-friendly interface for managing tweets, and admin privileges for comprehensive content control. Features include profile picture integration and tracking of user interactions such as followers and tweet likes.

#3 Resume GPT

Resume GPT is a Python/Django application that leverages the Google Docs and Drive APIs to automate resume creation and delivery. It dynamically generates resume templates based on user input and has successfully assisted over 30 individuals in creating their first resume.
